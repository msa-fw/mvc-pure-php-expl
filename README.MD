- Linux (Ubuntu 24.x)
- PHP 8.x
- Apache 2.2
- MySQL 5.x

--------------------------

**Install**

--------------------------

Run 

1. `php /path/to/install`
2. `php /path/to/cli db:migrate`
3. `php /path/to/cli server:run`

--------------------------

Write to crontab file line

```
* * * * * php /path/to/cli cron:run
```

--------------------------

Run `php /path/to/cli help` for call console commands reference
